Welcome to my page: Poetry is Protest

This is a small corner of the internet- my small corner. Not meant to be a big site or go viral (like all my other plans) but to be a place where I try and learn new things in HTML, CSS, JS and Interactive Fiction through my Twine stories.

There are many things I would like to share. But first and foremost, I will be sharing the things I have learned here. And the places that have helped me learn.

Tools:

  • Converting markdown to HTML: This is a neat little tool called Pandoc, which I use online.
  • IMGBB to host my image files. It is easy to make hot links there
  • Opendrive to host my audio files, things like background music, sound effects etc.
  • Additionally Jukehost is also a nice free option to host audio files for sound effects!

Resources:

  • Absolute beginner tutorials in HTML and CSS by Adam Hammond were very useful. As were his twinery tutorials. -W3 Schools for HTML and CSS documentation -Markdown reference guide and tutorial

If you are here and want to know more about me and my other, more finished work. I suggest you visit my website.

See you around! Stay healthy, stay happy and say NO to drugs! (and fascism).
